## 📜 JS 脚本示例

得益于内置的 JS 引擎，你可以像写网页一样轻松控制鼠标键盘。以下是一个简单的脚本示例：

```javascript
// 示例：循环点击屏幕上某个点，并在中间插入延迟
function Main(){
    // 每1s点击一次
    let count = 0
    while(true){
        Sleep(1000)
        MouseClick()
        Print(`当前点击第 ${++count} 次`)
    }
}
